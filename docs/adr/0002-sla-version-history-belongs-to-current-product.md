# SLA Version History Belongs to the Current Product

Status: Superseded by ADR 0003

SLA version-history HTML pages are modeled as exact Source Aliases of the current product's SLA Product Definition, rather than as separate Product Keys or CMS Business Payloads. The current route linked by the SLA Index remains the primary Source Location; this preserves historical provenance without publishing superseded contractual text as another current product page.
