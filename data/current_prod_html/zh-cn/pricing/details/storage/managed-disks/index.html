高级 SSD 托管磁盘

高级 SSD 托管磁盘是基于高性能固态驱动器 (SSD) 的存储，旨在支持 I/O 密集型工作负荷，具有非常高的吞吐量和低延迟。借助高级 SSD 托管磁盘，可设置永久磁盘，并配置其大小和性能特征。

高级 SSD 托管磁盘的总费用取决于磁盘的大小和数量，并且受到 出站数据传输 量的影响。这些磁盘大小提供不同的每秒输入/输出操作数 (IOPS)、吞吐量上限和每 GiB 每月价格。可以选择最符合所需存储大小、IOPS 和吞吐量的选项。可将若干个永久性磁盘附加到 VM 并对每个 VM 配置高达 PB 级的存储空间，使得每秒输入/输出操作达到 80,000 次、每个 VM 的每秒磁盘吞吐量达 1,600 MB，且读取操作的延迟低于 1 毫秒（针对可使用读取缓存的应用程序）。高级 SSD 托管磁盘由专门针对此类磁盘的 DS 系列、DSv2 系列、FS 系列和 GS 系列 VM 大小提供支持。在高级 SSD 托管磁盘上使用 Azure 共享磁盘时，每个额外装载的托管磁盘根据磁盘大小每月计费。

高级 SSD v2 是用于 Azure 磁盘存储的下一代高级 SSD 产品。高级 SSD v2 磁盘存储专为性能敏感型工作负载而设计，这些工作负载始终需要低平均读取和写入延迟以及高 IOPS 和吞吐量。此产品/服务非常适合支持数据密集型工作负载，例如 SQL Server、Oracle 数据库、Cassandra 和 Mongo DB。

使用高级 SSD v2，无需购买额外的存储容量即可预配所需的 IOPS 和吞吐量(请参阅下表)。因此，在部署事务密集型工作负载时，可以经济高效地缩放性能。高级 SSD v2 为任何磁盘大小提供 3,000 IOPS 和 125 MB/s 的基线性能，无需额外付费。磁盘存储容量可以介于至少 1 GiB 到最多 64 TiB 之间。

目前仅支持本地冗余存储(LRS)配置。

超级磁盘

超级磁盘是我们的下一代高性能固态硬盘 (SSD)，它具有可配置的性能特性，可提供最低延迟和一致的高 IOPS/吞吐量。超级磁盘可提供前所未有且高度可缩放的性能，其延迟低至亚毫秒级。客户可先选用较小的 IOPS 和吞吐量，再在工作负载占用更多 IO 时调整性能。

超级磁盘的总成本取决于磁盘大小及其性能配置，并受磁盘数量影响。

超级磁盘有不同大小规格，其每秒输入/输出操作数 (IOPS) 范围和吞吐量（MB/秒）范围均可配置，且按小时计费。可以选择最符合所需存储大小、IOPS 和吞吐量的选项。超级磁盘仅支持本地冗余存储 (LRS)。请参阅 Azure 存储复制页 ，了解有关冗余选项的更多详细信息。
