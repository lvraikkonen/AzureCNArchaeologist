    Azure Bastion

    无需 VM 上的公共 IP，即可保护到 Azure 虚拟网络中的 Azure VM 的无缝 RDP/SSH 连接

    借助 Azure Bastion 服务，无需 VM 上的公共 IP，无需任何其他客户端/代理或任何软件，即可直接在 Azure 门户中通过 RDP 和 SSH 安全、无缝地访问 Azure 虚拟网络中的 VM。在虚拟网络中预配 Azure Bastion 服务后，即可在同一虚拟网络中的所有 VM 上获得无缝 RDP/SSH 体验。有关详细信息，请参阅 文档 。
