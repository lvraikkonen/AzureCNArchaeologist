常见问题

全部展开
  • 什么是云连接服务（CCS）？

    云连接服务（CCS）是一站式的混合云解决方案（包括物理线路部署 + 网络代维）。 客户可以通过专线连接 Azure 数据中心和客户本地数据中心，部署快速、可靠、安全的混合云场景。

  • 使用云连接服务有什么好处？

    云连接服务提供专有网络连接服务，客户可以独享网络带宽。和通过公网 Internet 的连接相比，可靠性更高、带宽和延迟可预期且安全性更高。客户可以构建跨本地基础结构和 Azure 的应用程序，同时不会损害安全性或性能。

  • 云连接服务适用于哪些区域，对服务运营商有没有限制？

    支持由世纪互联运营的 Azure 北京数据中心（BJB）和上海数据中心（SHA）的接入服务。

  • 如何开始使用云连接服务？

    如需开通云连接服务请联系您的客户经理。

  • 云连接服务支持哪些传输速率？

    云连接服务提供 10Mbps、16Mbps、20Mbps、25Mbps、30Mbps、34MMbps、40Mbps、50Mbps、60Mbps、70 Mbps、80 Mbps、90 Mbps、100 Mbps、150Mbps ~ 2.5Gbps 带宽的专线接入。ExpressRoute 专用线路可用性至少为 99.95% 的服务级别协议前提是客户使用双线接入。

  • 云连接服务计费由哪几部分组成，什么时候开始计费？

    包括物理专线的一次性安装费和月租费两部分费用。不包括 ExpressRoute 产生的端口费用。
