通过 Azure 路由服务器，网络设备可以通过 Azure 虚拟网络动态交换路由信息。配置网络设备和 Azure ExpressRoute 以及 VPN 网关，自动从 Azure 路由服务器获取最新的路由信息，而不是手动与每个网络通信。
