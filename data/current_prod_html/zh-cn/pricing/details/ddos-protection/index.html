网络保护将具有固定的每月费用，其中包括对 100 个公共 IP 资源的保护。对于额外公共 IP 资源的保护将按月按资源收费。租户中的单个 Azure DDoS 防护计划可跨多个订阅使用。

价格
每月费用(包括对于 100 个公共 IP 资源的保护) ￥30,526/月
超额费用(超过 100 个公共 IP 资源) ￥305.1/每月每种资源

网络保护在虚拟网络(VNet)级别启用。在虚拟网络上启用网络保护后，该虚拟网络中所有受保护资源类型将自动受到保护。如果在受保护的虚拟网络中部署具有 WAF 的应用程序网关，则 WAF 无需额外付费 - 你将按照较低的非 WAF 费率为应用程序网关付费。如果使用网络保护来保护资源，则将包含 DDoS 攻击期间的任何横向扩展费用，并将向客户返还这些扩展资源的费用额度。

资源示例：

  • • Azure 防火墙以及包含 WAF 的 应用程序网关的数据处理(流入量/流出量)
  • • 虚拟机和 Azure Kuberentes 服务的横向扩展
  • • 网络带宽的数据流出 - 当受 DDoS 影响的应用建立出站连接时，在放大攻击期间发生。
  • • 横向扩展后端 PaaS 资源，如 SQL、CosmosDB、存储、应用服务等。

示例 4 包含具有 Azure 应用程序网关的 Azure Web 应用程序防火墙(WAF)的网络保护计划

假设你在由网络保护来保护的 VNet 中部署了具有 Azure Web 应用程序防火墙(WAF)的 Azure 应用程序网关。在此方案中，将按应用程序网关的基本定价收费，而不是按 WAF 定价对较高层级的应用程序网关收费。这意味着 WAF 使用不收取额外费用。此折扣适用于应用程序网关 WAF 和 WAF_v2 SKU。
