导入/导出 Import/Export

利用 Azure 导入/导出服务，你可使用硬盘驱动器在 Azure 中传入和传出大量数据。对于大数据集的移动来说，与通过 Internet 传输数据比较，导入/导出更快并且更加经济实惠。该服务使你能够通过硬盘驱动器到数据中心的安全传输和高速安全内部网络将数据传输到 Azure 中。

定价详细信息

设备处理：每个存储设备的处理费用为 ￥380。同一数据中心内的设备和 Azure 存储器之间的网络数据传输是免费的。常规存储事务是收费的。请查看存储价格详情页面。

设备返还运费：即将收取返还设备产生的相关运费。
