Azure Active Directory 现在更名为 Microsoft Entra ID

同步本地目录并启用单一登录

Microsoft Entra ID 是一个全面且高度可用的标识和访问管理云解决方案。它将核心目录服务、高级标识监管和应用程序访问管理相结合。Microsoft Entra ID 还提供了功能丰富的基于标准的平台，该平台支持开发人员基于集中的策略和规则为应用程序提供访问控制。

定价详细信息

Microsoft Entra ID 提供 3 个版本：免费版、高级版 P1、高级版 P2。在 Azure 帐户中包含免费版。如果你有 Azure 帐户，可以在线获取高级版 P1 和 P2。在此处登录进行购买。

高级版 P1

专用于支持对标识和访问管理具有更高要求的组织，Microsoft Entra ID 高级版添加了功能丰富的企业级标识管理功能，并让混合用户无缝访问本地和云功能。对于跨应用程序访问、自助标识和访问管理 (IAM) 以及云端安全的混合环境中信息工作者和标识管理员来说，此版本包含所有必需内容。
