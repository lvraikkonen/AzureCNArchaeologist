Microsoft Entra ID (formerly Azure AD)

Synchronize on-premises directories and enable single sign-on

Microsoft Entra ID is a comprehensive and highly available identification and access control cloud solution. It combines core directory services,advanced identity governance and application access management. Microsoft Entra ID also provides a standards-based platform with abundant functions which supports developers in providing access control for applications based on centralized policies and rules.
