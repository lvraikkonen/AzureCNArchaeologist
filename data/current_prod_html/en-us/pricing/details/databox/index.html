Azure Data Box

Appliances designed to simplify data movement in and out of Azure, both offline and online.

Data Box Disk is offline data transfer devices which are shipped between your datacenter and Azure. The Microsoft Azure Data Box Disk solution lets you send terabytes of on-premises data to Azure in a quick, inexpensive, and reliable way.

FAQ

  • How is my bill calculated?

    Customers are billed when the order has reached a terminal state (i.e the order has completed without errors).

  • For how long can I keep an Azure Data Box Disk at my premises?

    The Data Box Disk can be kept on-premises for a maximum time period of 45 days.

  • Is there a charge for a lost or damaged device?
