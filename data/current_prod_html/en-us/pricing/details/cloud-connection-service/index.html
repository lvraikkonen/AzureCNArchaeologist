FAQ

Expand all
  • What is Cloud Connection Service (CCS)?

    Cloud Connection Service (CCS) is a one-stop hybrid cloud solution (includes physical line deployment + network support). Customers can connect to Azure data centers and local customer data centers via dedicated lines to deploy fast, reliable, and secure hybrid cloud scenarios.

  • What are the advantages of using Cloud Connection Service?

    Cloud Connection Service provides a private network connection service, so that customers can enjoy exclusive network bandwidth. This is more reliable than a connection via the public Internet, with more predictable bandwidth and latency and higher security. Customers can build applications across on-premises infrastructure and Azure without compromising security or performance.

  • Which regions does Cloud Connection Service apply to, and are there any restrictions on service providers?

    Access services for the Azure Beijing Data Center (BJB) and Shanghai Data Center (SHA) operated by 21Vianet are supported.

  • How can I start using Cloud Connection Service?

    Please contact your account manager if you need to enable Cloud Connection Service.

  • Which transfer rates does Cloud Connection Service support?

    Cloud Connection Service provides dedicated line access at bandwidths of 10 Mbps, 16 Mbps, 20 Mbps, 25 Mbps, 30 Mbps, 34 Mbps, 40 Mbps, 50 Mbps, 60 Mbps, 70 Mbps, 80 Mbps, 90 Mbps, 100 Mbps, and 150 Mbps to 2.5 GBps. Customers use two-line access with a Service Level Agreement of at least 99.95% availability for ExpressRoute dedicated lines.

  • Which parts does Cloud Connection Service billing consist of, and when does billing start?

    Fees come in two parts and include a one-time installation fee and monthly fee for a physical line. Port costs generated by ExpressRoute are not included.
