Azure 更新管理器

Azure 更新管理器是一项服务，可帮助你跨 Azure、本地和其他云平台管理所有计算机(包括运行 Windows 和 Linux 的计算机)的更新。

Azure 更新管理器

对于已启用 Arc 的服务器，Azure 更新管理器将根据 ￥1.03/服务器/天的价格每天按比例计费，此价格相当于 ￥31.8/服务器/月(假设连接使用 31 天)。由 Azure Arc 启用的计算机只有在连接到 Arc 并由 Azure 更新管理器管理时才计费。 详细了解计算机何时由 Azure 更新管理器管理。

常见问题

全部展开
  • 如何为已启用 Arc 的服务器计算 Azure 更新管理器价格?

    对于已启用 Arc 的服务器，Azure 更新管理器费用为 $31.8/服务器/月(假定连接使用时间为 31 天)。按每日按 ￥1.03/服务器/日的比例值收费。启用 Azure Arc 的计算机仅在连接到并由 Azure 更新管理器管理的情况下才收费。

  • 启用 Arc 的服务器何时被视为由 Azure 更新管理器管理?

    当机器满足以下条件时，Azure 更新管理器将认为 Arc 启用的服务器由其管理：

    • Arc 在一天中的任何时间的连接状态。
    • 会触发更新操作(按需或通过计划作业进行修补、按需评估或定期评估)，或者其与计划相关联。
  • 是否存在未为 Azure 更新管理器收取已启用 Arc 的服务器的费用的情况?

    在以下情况下，不对使用 Azure 更新管理器管理的已启用 Arc 的服务器收费：

    • 如果机器启用了 Azure Arc 启用的扩展安全更新 (ESU)。
    • 托管启用 Arc 的服务器的订阅已启用 Microsoft Defender for Servers Plan 2。
  • 如果从自动化更新管理迁移到更新管理器，是否会向我收费?

    在以下情况下，不对使用 Azure 更新管理器管理的已启用 Arc 的服务器收费：

    在 LA 代理停用之前，如果客户将已启用 Arc 的服务器加入到启用了自动更新管理解决方案的同一订阅，则不用支付费用(截至 2023 年 9 月 1 日)。
  • 我是 Defender for Server 客户，使用 Azure 更新管理器支持的更新建议，即“应在计算机上启用定期评估”和“应在计算机上安装系统更新”。是否会向我收取 Azure 更新管理器费用?

    如果已购买 Defender for Servers 计划 2，则无需支付费用即可修正上述两项建议的不正常资源。但是，如果为 Arc 计算机使用任何其他 Defender for Server 计划，则 Azure 更新管理器将在每日按 ￥1.03/服务器比例对这些计算机收费。

  • Azure 更新管理器在 Azure Stack HCI 上是否收费?

    对于已启用 Azure 权益和 Azure Arc VM 管理的 Azure Stack HCI 群集上托管的计算机，不会对 Azure 更新管理器收费。了解详细信息
