常规

  • Azure 专用主机如何计费？

    配置主机后的每小时都会按即用即付费率计费，并且无论主机上运行的 VM 实例数量是多少，你都将为整个主机付费。任何其他费用（例如软件许可证）都将单独计费。

  • 软件许可证如何计费？

    软件许可证在 VM 级别根据使用情况单独计费。

  • 将工作负荷转换到 Azure 专用主机时，可以使用现有 Windows Server 和 SQL Server 许可证吗？

    由于硬件是专门为你提供的，因此适用于 2019 年 10 月 1 日之前获得的 Windows Server 和 SQL Server 许可证的外包软件管理条款，它允许你将现有许可证分配给 Azure 专用主机，就像为自己的数据中心的服务器分配许可证一样。

  • 可以在单个 Azure 专用主机上混合部署多种大小的实例吗？

    是的，可以在单个主机上混合部署不同大小的 VM。例如，可在同一主机上部署 D2s v3 和 D16s v3 VM。不能混合部署不同 VM 系列的 VM（例如 Dsv3 和 Esv3）。
