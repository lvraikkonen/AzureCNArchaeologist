    提供全面的混合安全管理和威胁防护

    更新时间：2025年09月03日

    Microsoft Defender for Cloud 在多云环境中提供从开发到运行时的全面云原生保护。Defender for Cloud 通过其免费层和增强的安全功能帮助你跨 Azure、其他云和本地保护资源。

    云安全态势管理 (CSPM)

    Microsoft Defender for Cloud 提供基础和高级的云安全姿态管理解决方案，以保护你的多云和混合环境。免费的基础 CSPM 提供持续评估、安全建议、安全分数和 Microsoft 云安全基准，覆盖 Azure、Amazon Web Services (AWS) 和 Google Cloud。

    Microsoft Defender CSPM 提供先进的安全状况功能，包括无代理漏洞扫描、攻击路径分析、集成数据感知安全状况、代码到云上下文化和智能云安全图。定价取决于云大小，仅基于服务器、存储帐户和数据库计数进行计费。

    此外，它还包括 DevOps 安全功能，使安全团队能够跨多管道环境管理 DevOps 安全性。
