Microsoft Entra 域服务 (Azure AD DS)

用于虚拟机和目录感知应用程序的域服务

Microsoft Entra 域服务 (Azure AD DS)提供可缩放的高性能托管域服务，如域加入、LDAP、Kerberos、Windows 集成身份验证和组策略。单击按钮后，管理员即可为 Azure 基础结构服务中部署的虚拟机和目录感知应用程序启用托管域服务。通过保持与 Windows Server Active Directory 的兼容性，Azure Active Directory 域服务使管理员能够轻松地将旧的本地应用程序迁移到云，并集中管理 Azure Active Directory 中的所有应用程序和所有标识。

支持和 SLA

Microsoft Entra 域服务 (Azure AD DS)的技术支持可通过 Azure 支持获取，计费和订阅管理支持免费提供。

SLA - 对于托管域中的用户帐户的域身份验证、到根 DSE 的 LDAP 绑定或记录的 DNS 查找，我们将保证至少成功完成 99.9% 的 Microsoft Entra 域服务 (Azure AD DS)请求。了解有关SLA的详细信息。预览版期间，此 SLA 不适用于资源林企业层。
