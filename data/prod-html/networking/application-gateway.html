常见问题

全部展开
  • 不足一小时的使用是否会按照一个小时计费？

    是的。不足一小时将按照一小时计费。

  • 是否只为只有一个实例的应用程序云服务提供服务级别协议？小型应用程序网关呢？

    否。小型规模的应用程序网关主要用于开发/测试用途。在小型应用程序网关上运行生产工作负载可能会超过小型实例的处理能力。应用程序网关支持在同一个云服务中部署一个或多个实例。由于缺乏高可用性，不建议运行只有一个实例的应用程序网关云服务。

  • 什么是容量单位？

    容量单位会测量在固定成本之外计费的基于消耗的费用。容量单位费用也按小时或部分按小时计算。容量单位有 3 个维度：计算单位、持久连接和吞吐量。计算单位测量的是所使用的处理器容量。请参阅我们的 文档页 。

如有任何疑问或需要帮助，请访问 Azure 支持 选择自助服务或者其他任何方式联系我们获得支持。
