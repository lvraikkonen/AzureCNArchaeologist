  • 现有工作区和资源将发生什么情况？

    你的现有工作区将默认为基本版。你将仍有权访问实验、计算目标、存储和模型等各项现有资源。

  • 如何在基本版和企业版工作区之间切换？

    要将工作区从基本版升级为企业版，请访问 Azure 门户。你可以： 1. 转到工作区列表，再指定要升级的工作区，选择“升级”开始此过程。或 2. 在 Azure 门户中，创建新的工作区，然后将“工作区类型”指定为“企业版”。目前不支持通过 SDK 或 CLI 将工作区升级为企业版。 你将无法将工作区从企业版降级为基本版。有关升级的详细信息，请访问 机器学习文档 。

  • 是否要对 Azure 机器学习上的训练和推理模型计费？

    无论是基本版还是企业版，客户都仍需支付所使用的 Azure 资源的费用。有关 VM 价格，请参阅 此处 。企业版为预览版时，不收取机器学习附加费，但在企业版正式发布后，将按此页所示收取机器学习附加费。

  • 如何对 Azure 机器学习设计器计费？

    客户可通过 Azure 机器学习访问 Azure 机器学习设计器。使用设计器功能无需额外成本。
