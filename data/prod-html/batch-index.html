批处理 （预览） Batch

云规模的作业计划和计算管理

Azure 批处理提供作业计划和群集管理，允许应用程序或算法大规模并行运行。

对 Batch 本身不收费，仅对运行 Batch 作业所用的基础计算和其他资源收费，包括适用的软件许可费用。对于计算，批处理可以使用云服务、Linux 虚拟机或 Windows 虚拟机。收取标准的计算费率，可在下面的内容中查看。下文中还提供了批处理图形和呈现的软件许可费用。

  • Windows 虚拟机
  • Linux 虚拟机
  • 云服务
  • 计算优化
  • 内存优化

计算优化: 高 CPU 与内存之比。适用于中等流量的 Web 服务器、网络设备、批处理和应用程序服务器。

Fv2 系列

Fv2 系列虚拟机支持每个 vCPU 2 GiB RAM 和 8 GB 本地临时存储 (SSD)，并针对计算密集型工作负荷进行了优化。F 系列为超线程机，基于 2.7 GHz Intel Xeon® Platinum 8168 (SkyLake) 处理器，该处理器使用 Intel Turbo Boost Technology 2.0 可实现高达 3.7 GHz 的时钟速度。这些虚拟机适合批处理、Web 服务器、分析和游戏等方案。
