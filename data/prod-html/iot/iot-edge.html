Azure IoT 边缘 Azure IoT Edge

将云智能和分析扩展到边缘设备

Azure IoT Edge 将云服务部署到 IoT 设备，让你可以在边缘设备本地处理除数、运行分析并获取见解。这是通过部署人工智能、Azure Functions 和 Azure 流分析等云服务，并从 Azure IoT 中心管理边缘设备实现的。还可部署自定义代码，从而在边缘设备上部署自定义的复杂业务逻辑。Azure IoT Edge 可在云和边缘实现代码对称，简化开发工作，让开发者能够专注于构建对最终用户而言有用的功能，而无需管理基础结构。

定价详细信息

需要使用 Azure IoT 中心，才能安全管理通过 Azure IoT Edge 部署到边缘的设备和服务。Azure IoT Edge 可在装有 Windows 或 Linux 的任意硬件上运行。将 Azure IoT Edge 部署到内存和其他资源充足的 IoT 设备，通常是网关类设备。

Azure IoT 中心 详细信息 定价

Azure IoT Edge 可免费使用。通过Azure IoT Edge，可在边缘运行多项 Azure 服务。在 IoT Edge 运行的 Azure 服务将根据其各自的定价进行计费。

  • IoT Edge 的使用如何计费？

    Azure IoT Edge 本身是免费的，但需要使用 Azure IoT 中心来安全管理设备和边缘部署。通过选择通过 IoT Edge 使用 Azure 服务，则将根据在边缘使用特定服务的计费模型进行计费。例如，如果选择在 IoT Edge 上运行 Azure 流分析，则将根据 IoT 中心使用情况和 Azure 流分析边缘定价进行计费。
