Azure Private Link pricing

Improved security and routing for your Azure service resources

Updated on：2025/11/05

With Azure Private Link, Azure customers can render and consume services privately on Azure Platform. Services can be Azure PaaS services such as Storage, SQL and so on, Marketplace Service (Service Provider rendering his service on Azure Platform) or Customer’s own service. For more information, please refer to the documentation .

FAQ

Expand all
  • How will partial hours be charged?

    Partial Hours will be charged as full hours.

  • How will billing work when accessing private endpoint from peered network?

    While accessing Private Endpoints from peered Network, you will only be charged for Private Link Premium. You will not be charged for Peering.

  • Will I be charged for Data transfer in addition to Private Link?

    Yes, above prices are premiums for Private Link capability. Data transfer pricing will apply to data.
