FAQ

Expand all
  • Will usage of less than one hour be billed as one hour?

    Yes, usage of less than one hour will be billed as one hour.

  • Is a Service Level Agreement offered for an application cloud service with only one instance? What about small application gateways?

    No. The small-sized Application Gateway is mainly used for development/testing purposes. Running production workloads on a small-sized Application Gateway may exceed the capacity of small instances. Application Gateway supports deploying one or several instances in the same cloud service. Due to a lack of high availability, running the Application Gateway service with only one instance is not recommended.

  • What is Capacity Units?

    Capacity Units measure consumption-based cost that is charged in addition to the fixed cost. Capacity unit charge is also computed hourly or partial hourly. There are three dimensions to capacity unit - compute unit, persistent connections, and throughput. Compute unit is a measure of processor capacity consumed. Please refer to our documentation page .
