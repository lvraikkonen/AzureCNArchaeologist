FAQ

Expand All
  • How often can I change the node type I have deployed?

    Customers can change the node type deployed as needed.

  • How does the usage of Power BI Embedded show up on my bill?

    Power BI Embedded bills on a predictable hourly rate based on the type of node(s) deployed. Actual usage is computed to the second and billed hourly. For example, if an instance exists for 12 hours and 15 minutes in a month, your bill will show usage of 12.25 hours. If your instance is only active for 6 minutes, your bill will show usage of 0.1 hour.

  • Do I need to purchase any additional licenses to publish BI into a Power BI Embedded node?

    Yes, users who publish BI content need to be licensed with Power BI Pro.

  • Do users who view BI published through Power BI Embedded need to be licensed separately?

    No, users viewing content do not need Power BI licenses assigned to them.

  • What happens when I pause my service?

    When the service is paused the embedded content will not load, and you will not be charged for the service.

  • How does Power BI Embedded relate to the Power BI service?

    Power BI Embedded is for developers who build applications and want to use a third-party BI offering to visualize application data instead of building it themselves. Developers embed dashboards and reports into applications using Power BI Embedded. Power BI, on the other hand, is a software-as-a-service analytics solution that gives organizations a single view of their most critical business data.

  • Can Power BI Embedded be used to create internal applications?

    Yes, however if you plan to use Power BI SaaS embedding you will need to use Power BI Premium through office (EM1-EM3; P1-P3).
